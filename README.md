# OS Simulation Game

## Prerequisites

* Python 3.10
* [pipenv](https://pypi.org/project/pipenv/)
* Create an empty `.venv` directory at the root of the project

## Usage

* Install dependencies: `pipenv install`
* Run program: `python src/main.py`

```bash
pipenv shell
python main.py
```
